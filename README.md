# 智慧戒斷盒 - Flask 版本

一個幫助你遠離手機干擾、專注工作的智慧鎖定系統。

## 功能特色

- 🔒 **鎖定控制** - 設定鎖定時間，強制遠離手機
- 📊 **成果儀表板** - 追蹤專注時數與成就統計
- 🚨 **AI 緊急解鎖** - 智慧判斷緊急狀況
- 🎨 **藍白配色** - 清新專業的使用者介面

## 安裝步驟

1. 安裝依賴：
\`\`\`bash
pip install -r requirements.txt
\`\`\`

2. 啟動應用：
\`\`\`bash
python app.py
\`\`\`

3. 在瀏覽器開啟：
\`\`\`
http://localhost:5000
\`\`\`

## 技術架構

- **後端**: Flask (Python)
- **前端**: HTML5 + CSS3 + JavaScript
- **圖表**: Chart.js
- **圖示**: Lucide Icons

## 開發說明

- `app.py` - Flask 主應用程式
- `templates/` - HTML 模板 (Jinja2)
- `static/css/` - 樣式表
- `static/js/` - JavaScript 腳本
